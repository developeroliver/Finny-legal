Politique de Confidentialité

Dernière mise à jour : 14 juillet 2025
Comment nous protégeons vos informations financières

Introduction

Bienvenue sur Finny. Cette Politique de Confidentialité décrit comment nous collectons, utilisons et protégeons vos informations lors de l'utilisation de notre application de gestion budgétaire. Nous nous engageons à protéger vos données financières et à assurer la transparence sur la façon dont vos informations sont traitées.

Authentification via Firebase

Finny utilise Firebase Authentication via Apple Sign In pour permettre aux utilisateurs de se connecter de manière sécurisée et d'accéder aux fonctionnalités personnalisées.

Ce que nous collectons via Firebase :

  • Adresse e-mail

Ces données sont utilisées uniquement pour les fonctionnalités de l'application, telles que :

  • Vous permettre de vous connecter de manière sécurisée
  • Stocker et synchroniser vos données budgétaires entre appareils
  • Sauvegarder vos catégories de dépenses et revenus personnalisés
  • Conserver vos abonnements et transactions enregistrés

Nous n'utilisons pas votre nom ou e-mail à des fins publicitaires, et ces données ne sont jamais partagées avec des tiers à des fins marketing.

Gestion des Abonnements et Achats via RevenueCat

Finny utilise RevenueCat pour fournir des abonnements premium, garantissant une expérience fluide et sécurisée.

Ce que RevenueCat collecte en notre nom :

  • Un identifiant utilisateur anonyme généré aléatoirement
  • Détails des transactions (ex : date d'achat, ID produit, statut d'abonnement, informations de renouvellement)

Ces données sont utilisées pour :

  • Activer et gérer les fonctionnalités premium (rapports avancés, synchronisation illimitée, etc.)
  • Prévenir la fraude par validation des reçus
  • Générer des analyses agrégées pour nous aider à améliorer l'application
Important : RevenueCat ne reçoit pas votre nom d'utilisateur ou e-mail Firebase. L'identifiant utilisateur RevenueCat est anonyme et non lié à votre identité personnelle.

Données Financières

Finny traite vos informations financières avec le plus grand soin :

Types de données collectées :

  • Montants de revenus et dépenses
  • Catégories de transactions
  • Informations sur les abonnements
  • Données budgétaires personnelles

Utilisation :

  • Calculer vos soldes et statistiques financières
  • Générer des rapports et analyses personnalisés
  • Fournir des recommandations budgétaires
  • Synchroniser vos données entre appareils
Important : Nous ne collectons jamais d'informations bancaires réelles (numéros de compte, mots de passe, etc.). Toutes les données financières sont saisies manuellement par vous et restent sous votre contrôle.

Comment nous utilisons vos données

  • Firebase : Pour l'authentification et la synchronisation de vos données budgétaires personnalisées
  • RevenueCat : Pour gérer les abonnements premium, activer les droits, valider les achats et analyser le comportement d'abonnement agrégé
  • Traitement local : Calculs budgétaires et génération de rapports sur votre appareil

Stockage de vos données

  • SwiftData : Toutes vos données financières (revenus, dépenses, catégories, budgets) sont stockées localement sur votre appareil
  • Synchronisation Cloud : Si vous vous connectez, vos données sont synchronisées de manière sécurisée via Firebase pour être accessibles sur tous vos appareils
  • Suppression : Vous pouvez supprimer toutes vos données financières lorsque vous supprimez votre compte

Vos choix

  • Vous pouvez utiliser les fonctionnalités principales de Finny gratuitement
  • Votre e-mail n'est stocké que si vous choisissez de vous connecter pour la synchronisation
  • Les données d'abonnement sont traitées de manière anonyme via RevenueCat
  • Vous contrôlez entièrement vos données financières et pouvez les supprimer à tout moment

Sécurité

Nous mettons en place des mesures de sécurité appropriées pour protéger vos informations financières :

  • Chiffrement des données en transit et au repos
  • Authentification sécurisée via Firebase
  • Stockage local sécurisé sur votre appareil
  • Pas de partage de données avec des tiers non autorisés

Conservation des données

Nous conservons les données d'authentification et d'achat tant qu'elles sont nécessaires pour fournir les fonctionnalités de l'application ou répondre aux obligations légales. Étant donné que RevenueCat utilise des identifiants anonymes, nous ne pouvons pas supprimer les enregistrements d'achat individuels. Les données d'identité stockées par Firebase (ex : e-mail) peuvent être supprimées sur demande.

Modifications de cette Politique de Confidentialité

Nous pouvons mettre à jour cette Politique de Confidentialité périodiquement. Tout changement sera reflété ici, et la date de "Dernière mise à jour" sera révisée en conséquence. Nous vous notifierons des modifications importantes via l'application.
